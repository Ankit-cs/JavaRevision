Automatic Type Promotion :

int i=(a*b)/c as it will go beyond the range of byte so java automatically promote byte to int
Rules for this Promotion
1->Byte ,short,char values are promoted to int
2->if one operand is long the whole expression will become long
3->if one operand is double then entire expression is double
